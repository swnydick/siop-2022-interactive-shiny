<h3><span style="color: #006550;"><strong>App 6: Adding Javascript Components</strong></span></h3>
<p>This app will demonstrate how you can add a simple javascript component within your application using the shinyjs package.&nbsp; Specifically, we'll use shinyjs to add a reporting button that will hide the reporting sentences and table. More information and demonstrations can be found here : <a href="https://deanattali.com/shinyjs/">shinyjs R package | by Dean Attali | shinyjs</a> .&nbsp;&nbsp;</p>
<p>Specifically, the following new shiny functions will be introduced :</p>
<p style="padding-left: 40px;"><span style="color: #494849;">actionButton( ), toggle( ), toggleClass( ), div( ), html( )</span></p>
